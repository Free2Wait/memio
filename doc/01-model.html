Model

In order to generate code, you must build some models. Here's all you need to know about them.

  • Argument
  • ArgumentCollection

See also next readings.

Argument

A method's single argument. It has a Type and a name:

<?php

namespace Gnugat\Medio\Model;

class Argument
{
    public function __construct(Type $type, $name);
}

The Type model is responsible of knowing if the argument can be type hinted:

<?php

namespace Gnugat\Medio\Model;

class Type
{
    public function __construct($name);
}

Note: a type is considered as being an object if the given name is different than string, bool, int, double, callable, resource, array, null, mixed.

ArgumentCollection

Methods can have 0 to many arguments, this model takes care of managing those:

<?php

namespace Gnugat\Medio\Model;

class ArgumentCollection
{
    public function add(Argument $argument);
}

Note: ArgumentCollection will rename added arguments in case of name collision, by suffixing them with a number.
